## 七、问题与处理记录

### 7.1 M4 里程碑延期

真实 Elasticsearch 搜索尚未实现，不能用 Mock 数组过滤替代验收。调整计划为 7 月 20–21 日完成索引对象、Repository、同步服务和高亮查询后重新验收。

### 7.2 Elasticsearch 与 Spring Data 版本兼容

服务端固定为 `7.17.21`，需要核对当前 Spring Boot 3.2.5 所带客户端兼容性；在完成最小连接测试前不引入复杂查询封装。

### 7.3 IK 分词器安装一致性

IK 插件版本必须与 ES 服务端一致。若团队环境无法稳定安装，先使用标准分词器完成接口闭环，再补充中文分词能力。

### 7.4 基础功能技术债务累积

认证、职位、简历和文件模块存在接口骨架先行、真实业务滞后的问题。周末复盘需限制新增页面，优先补齐阻断真实联调的 Service 层。
